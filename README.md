# po-diff

Compare PO/TS translation files and show differences.

Built with Python CLI. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install po-diff
```

### Fedora/RPM
```bash
sudo dnf install po-diff
```
